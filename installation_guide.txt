1. Upload the "application", "js", "system", "licence.txt" and "mystyles.css" in the root of a webfolder.
2. Create a CodeIgniter database, configure it right in the application/config file and upload there the SQL code of the codeigniter.sql file
3. Go to <path>/index.php/. E.G. http://localhost/CodeIgniter_2.1.2/index.php/
